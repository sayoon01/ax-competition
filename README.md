# 대댐청

## 📌 개요
대청댐 조류 발생 예측을 위해  
조류, 수질, 기상, 수문 데이터를 통합하여  
모델 학습용 데이터셋을 구축한 프로젝트

---

## 📊 사용 데이터

1. 조류 데이터 
2. 수문 데이터
3. 기상 데이터 
4. 댐 운영 데이터 

---

## ⚙️ 전처리 과정

- 주 단위 조류 데이터를 **일 단위로 확장 (exponential growth 기반)**
- 수질 데이터는 **선형 보간 (interpolation)**
- 기상 및 수문 데이터는 **날짜 기준 merge**
- 중복 컬럼 정리 (_x, _y 처리)
- 발령단계 생성 (total_cyano 기준)

---

## 📁 최종 데이터셋 

- 각자 사용하기 편한대로 3개로 나눠서 사용해도 됨

- Feature Engineering (lag, rolling)
